# How to change languages in a Grails app?

This repository contains the worked sample app for the [How to change languages in a Grails app?](https://grails.apache.org/guides/grails_i18n/4/guide/index.html) guide on the Apache Grails site.

## Run the example

`./gradlew bootRun`

## Maintained at apache/grails-static-website

The guide narrative is maintained at https://github.com/apache/grails-static-website. Changes to the guide content go there; this repo holds the worked sample app only.
